# humanizer-zh

Polish Chinese drafts while preserving meaning, evidence, and author voice.

## Scope

Use this skill for Chinese text polish, de-AI editing, and publication cleanup.

## Workflow

1. Preserve the author's meaning and evidence.
2. Remove generic filler, empty transitions, and AI-sounding conclusions.
3. Keep claims grounded in the supplied draft.
4. Before finalizing, check that no new claim, citation, or unsupported transition was introduced.

## Output rules

Return only the polished Chinese text unless the user asks for commentary. Avoid adding a generic conclusion unless the source draft already has one.
